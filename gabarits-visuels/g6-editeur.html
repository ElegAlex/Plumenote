Infrastructure INF-VPN-001 · Type : Procédure · Créé par Mohamed Zemouche · 15 février 2026

Cette procédure décrit la configuration du VPN Always-On pour les postes utilisateurs DSI et des services de la CPAM 92, dans le cadre de la migration Windows 11 (MUDT).

Contexte et pré-requis

L'objectif est d'obtenir une connexion VPN qui démarre automatiquement à l'ouverture de session utilisateur, sans intervention manuelle, et qui couvre tous les flux DSI quel que soit le réseau source.

Avant de commencer, vérifiez les pré-requis ci-dessous :

  • Le poste doit être joint à l'annuaire national (CPAM*.CRAMIF.LOCAL).
  • Le certificat machine AD-CS 2024 doit être présent dans le magasin local.
  • Windows 11 23H2 minimum.
Astuce. Si votre poste a déjà été migré via l'outil MUDT, l'intégralité des pré-requis est déjà remplie.

Installation du client

Le client VPN est distribué par l'outil de déploiement MUDT :

PowerShell
# Installation silencieuse
$installer = "\\srv-deploy\mudt$\vpn\CPAMVPNClient-2.4.1.msi"
Start-Process msiexec.exe "/i $installer /quiet" -Wait
